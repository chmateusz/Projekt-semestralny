# DNA Motif Analyzer

Aplikacja desktopowa w Pythonie do analizy motywów sekwencyjnych w DNA.

## Opis projektu

Program umożliwia:
- wczytywanie sekwencji DNA z plików FASTA/TXT,
- pobieranie sekwencji z bazy NCBI na podstawie accession ID,
- wyszukiwanie jednego lub wielu motywów DNA,
- analizę jednej lub dwóch sekwencji,
- porównanie liczby wystąpień motywów,
- segmentację sekwencji na biny i analizę rozmieszczenia motywów,
- interaktywną wizualizację pozycji motywów na osi sekwencji,
- eksport raportu PDF.



## Funkcje programu

- wczytanie sekwencji z pliku FASTA/TXT,
- wyszukiwanie motywu i określanie jego pozycji,
- obliczanie liczby wystąpień,
- segmentacja sekwencji przy użyciu NumPy/Pandas,
- wizualizacja wyników,
- proste GUI.
- pobieranie sekwencji z NCBI,
- obsługa wielu motywów jednocześnie,
- porównanie dwóch sekwencji,
- interaktywne wskazywanie pozycji motywów na wykresie,
- eksport zbiorczego raportu PDF.

## Wymagania

Program wymaga Pythona 3.x oraz bibliotek wymienionych w pliku `requirements.txt`.
